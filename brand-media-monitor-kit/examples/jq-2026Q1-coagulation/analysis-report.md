# 九强生物 2026Q1 凝血相关品牌监测报告

## Executive Summary

在 `2026-01-01` 到 `2026-03-31`、且主题限制为“凝血相关品牌活动或报道”的前提下，公开可稳定回收的九强生物相关内容较少。

这并不代表系统失败，而是说明：

- 这一主题在本次窗口内并非公开传播强主线
- 或者相关内容更多分布在不易公开回收的渠道

## Official Output Review

- 官网层：
  - 可检索到窗口外凝血相关内容，如 `2025-01-06` 的血栓四项内容
  - 但在本次 `2026Q1` 窗口内未稳定命中
- 公众号层：
  - 本轮公开层未稳定回收到符合范围的原文链接

## External Media Pickup

- 命中 2 条百度补面结果：
  - 上海证券报人物报道
  - 新浪财经资讯
- 这两条都属于外部感知面补充，不足以证明九强在该窗口进行了强力凝血传播

## Risk / Sentiment

- 未发现明确负面
- 但“无负面”不等于传播结构健康
- 更重要的结构性信号是：
  - 在指定窗口里，凝血主题的公开传播面偏弱

## Cooperation Recommendations

- 行业媒体池建议保持固定，便于横向比较不同品牌
- 后续若要提升准确率，优先增强公众号原文回收链

## Next-Stage Content Recommendations

- 如果品牌目标是强化“九强在凝血上的存在感”，下一阶段应增加：
  - 凝血产品 / 菜单 / 方案表达
  - 可被媒体直接引用的节点型内容
  - 将业务能力转化为外部可感知品牌主张

## Monitoring Caveats

- 本次结果严格遵循主题与时间限制
- 低命中与零命中是有效监测结果
- 不应为了让报告看起来更丰满而自动放宽范围
