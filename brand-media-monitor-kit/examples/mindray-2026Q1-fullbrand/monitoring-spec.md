# Monitoring Spec

## Monitoring Goal

监测迈瑞医疗在 `2026-01-01` 到 `2026-03-31` 期间，围绕品牌宣传稿件与媒体端投放内容的公开传播表现。

重点识别：

- 官方是否持续输出品牌主叙事
- 外部媒体是否形成连续放大
- 传播节奏是否具备活动设计特征

## Risks / Caveats

- 公众号历史导出依赖渠道可访问性，不保证绝对全量。
- 公众号池若不走真实登录态，公开层可回收率有限。
- 外部媒体补面不能替代官网主证据。
