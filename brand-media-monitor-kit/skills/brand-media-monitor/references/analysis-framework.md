# Analysis Framework

## 1. Corpus Audit

- total official posts
- total external media mentions
- monthly distribution
- source-origin split:
  - official original
  - official-sourced repost
  - media original interpretation
  - search/web supplement

## 2. Topic Mix and Topic Logic

- category distribution
- dominant topic lines
- whether content serves:
  - education
  - activity exposure
  - brand-image building
  - product/menu expansion
  - ecosystem cooperation
  - employer branding
- topic gaps and over-concentration

## 3. Publishing Rhythm

- monthly rhythm
- weekday / time pattern if visible
- activity-node concentration
- whether there is a release sequence:
  - official first, media later
  - preview, exposure, interpretation
  - event, interview, brand upgrade

## 4. Official vs Media Amplification

- which official narratives get amplified
- which media roles appear:
  - quick news
  - product observation
  - interview / viewpoint
  - finance / capital-market layer
- which media repeatedly cover the brand

## 5. Quality and Trust Pattern

- professional authority
- proof-chain quality
- readability / title force /传播感
- compliance and overstatement risk
- whether the brand is building trust through:
  - experts
  - data / results
  - installed-base proof
  - ecosystem cooperation
  - public welfare / social responsibility

## 6. Risk

- explicit negative media
- weak-signal risk:
  - over-concentration in one topic type
  - low diversity of media
  - lack of application stories
  - too much exposure but not enough educational translation
  - too much brand self-praise but weak scenario proof

## 7. Recommendation

- keep / expand / pause media cooperation
- content themes to add next
- issues to keep monitoring
- content-column suggestions
- release-rhythm suggestions
- official/media coordination suggestions
