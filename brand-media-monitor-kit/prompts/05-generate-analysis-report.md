# Prompt 05: Generate Analysis Report

```text
请根据原始数据表输出品牌内容传播分析报告。

必须回答：
1. 在指定时间里，这个品牌真实发布了什么
2. 哪些是官方主线，哪些被外部媒体放大
3. 有无明确负面或潜在风险
4. 即使没有负面，内容结构是否仍有弱点
5. 哪些媒体值得继续合作
6. 下一阶段品牌内容应该补什么

禁止：
- 只写“无明显负面”
- 只做数量统计
- 把低样本窗口写成系统失败

输出结构建议：
1. Executive Summary
2. Corpus and Evidence Notes
3. Topic Matrix and Topic Strategy
4. Publishing Rhythm and Activity Strategy
5. Official vs Media Amplification
6. Risk and Structural Weakness
7. Cooperation Recommendations
8. Next-Stage Content Recommendations
9. Monitoring Caveats
```
