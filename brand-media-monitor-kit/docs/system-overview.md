# System Overview

## Positioning

`Brand Media Monitor Kit` 是通用品牌内容监测系统，不是单次搜索脚本。

它服务于：

- 自有品牌季度复盘
- 竞品内容传播分析
- 活动后传播效果回看
- 专题型舆情监测

## Core Inputs

- 品牌名称与关键词
- 时间窗口
- 主题限制
- 官方渠道清单
- 外部媒体清单

## Core Outputs

1. `raw-table.csv`
2. `analysis-report.md`

## Design Principles

- 官方与媒体证据分层
- 公众号层优先一手证据
- 低命中和零命中也保留
- 不为凑样本而扩大范围
- 报告必须输出可执行建议

## What "Good" Looks Like

一份合格报告至少应该说明：

- 品牌主叙事是什么
- 媒体如何放大
- 节奏是否有设计
- 风险与空位在哪里
- 下一步怎么做
