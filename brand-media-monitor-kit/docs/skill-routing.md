# Skill Routing

这套系统是多技能串联，不是单技能完成。

## Required Routing

1. `brand-media-monitor`
- 任务编排
- 字段规范
- 报告口径

2. `agent-reach`
- 公众号与平台搜索入口
- 公众号原文链接发现

3. `wechat-article-extractor`
- 公众号文章结构化解析
- 标题、账号、发布时间补全

4. `web-access`
- 官网与媒体站点补抓
- 搜索补面与真实浏览器场景

## Optional Writing Routing

5. `c-account-manager`
- 将监测结果转为可写素材包

6. `dan-koe-coach-v2` / `linyue-c-wechat-writer`
- 将系统方法写成可分享文章

## Recommended Sequence

1. Define spec
2. Build channel registry
3. Collect by channel
4. Normalize raw table
5. Generate analysis report
6. Convert to action recommendations
7. Optional article writing
